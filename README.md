# Curso: Ciência de Dados
---------------------------------

## | Requisitos Funcionais (RF):
    - Calcular notas e médias.
    - Identificar quem se encontra em Recuperação.
    - Identificar Top Student (Best Grades).
    - Processar uma estrutura de dados composta por Listas e Tuplas.

## | Requisitos Não Funcionais (RNF):
    - Forma prática e clara.
    - Segurança.
    - Organização.
    - Modular.

## | Regras de Negócio (RN):
    - A média do aluno deve ser calculada a partir das notas cadastradas.
    - Alunos com média abaixo de 7.0 entram em recuperação.
    - O aluno com maior média será identificado como Top Student.
    - O sistema deve percorrer os dados de forma eficiente.
    - O relatório final deve apresentar: nome do aluno, média e situação.
    - O relatório final deve estar em .txt.
